Amity Scholarship Login Portal

A modern, responsive login page for the Amity University Scholarship Portal, created with HTML, CSS, and JavaScript.<br>

This project is designed for students to easily log in to their scholarship dashboard in a clean, secure, and mobile-friendly environment.<br>
✨ Key Highlights<br>
🎨 Elegant Design – Minimalist white-and-blue color scheme matching Amity’s brand identity.<br>

📱 Fully Responsive – Adapts to any screen size, from large monitors to small smartphones.<br>

🔄 Simulated Login Animation – Displays a loading spinner to mimic server authentication.<br>

♿ Accessibility First – Semantic HTML, visible focus outlines, and labeled form controls.<br>

🌙 Dark Mode Placeholder – Toggle button ready for future functional dark mode.<br>

🔒 Ready for Backend Integration – Frontend form designed for secure API connection.<br>
